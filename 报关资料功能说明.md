# 报关资料生成功能说明

## 功能概述

在SeekTrace CRM系统中成功添加了**报关资料生成功能**，允许用户从商机直接生成完整的报关资料Excel文件。

## 功能特点

### 1. 一键生成
- 从商机卡片或商机详情页点击"生成报关资料"按钮
- 自动获取商机、客户和关联的PI/CI单据信息
- 智能填充报关所需的各项数据

### 2. 完整的报关资料包
生成的Excel文件包含7个工作表：
1. **报关单** - 海关出口货物报关单，包含发货人、收货人、运输方式、货物明细
2. **申报要素** - HS编码、品牌、型号、品牌类型等海关申报要素
3. **发票 (Invoice)** - 商业发票，包含货物描述、数量、单价、总额
4. **箱单 (Packing List)** - 装箱单，包含箱数、毛重、净重
5. **合同 (Contract)** - 销售合同
6. **委托书** - 代理报关委托书
7. **填制规范** - 报关单填写说明文档

### 3. 数据自动填充
- **发货人信息**：从卖方信息自动填充
- **收货人信息**：从客户资料自动获取
- **货物信息**：从商机和PI/CI单据获取产品、数量、价格
- **运输信息**：从贸易条款自动推导
- **支付信息**：从客户默认条款或PI/CI获取

### 4. 可视化预览
- 生成前显示完整的报关资料预览
- 分模块展示：基本信息、发货人、收货人、运输、货物明细、支付条款
- 表格形式展示货物清单，计算总金额
- 提示包含的文档类型和注意事项

### 5. 优雅的用户界面
- 现代化的卡片式布局
- 清晰的信息分组和层次
- 使用图标增强可读性（📋 📦 🚢 📝 💰）
- 响应式设计，适配不同屏幕

## 使用场景

### 适用阶段
商机处于以下阶段时可以生成报关资料：
- ✅ 已报价
- ✅ 样品
- ✅ 谈判
- ✅ 成交

### 前置条件
1. 商机必须关联客户
2. 建议先生成PI或CI单据（系统会自动关联数据）

## 使用方法

### 方式一：从商机卡片
1. 在商机管道视图中，找到目标商机
2. 点击商机卡片上的"生成报关资料"按钮
3. 在预览窗口中检查信息
4. 点击"导出 Excel"按钮下载文件

### 方式二：从商机详情
1. 点击商机卡片打开详情抽屉
2. 点击"生成报关资料"按钮
3. 在预览窗口中检查信息
4. 点击"导出 Excel"按钮下载文件

## 技术实现

### 后端 API
- **生成接口**：`POST /api/deals/:dealId/generate-customs`
  - 根据商机ID生成报关资料数据结构
  - 自动关联客户和PI/CI单据信息
  
- **导出接口**：`POST /api/customs-documents/export`
  - 将报关资料数据导出为Excel文件
  - 使用xlsx库生成多工作表文件

### 前端功能
- 在商机卡片和详情页添加按钮
- 实现报关资料预览模态框
- 实现Excel文件下载功能
- 优化的CSS样式和用户体验

### 数据模型
扩展了TypeScript类型定义：
- `CustomsDocument` - 报关资料数据结构
- `TradeDocumentItem` - 扩展了报关相关字段（品牌、型号等）

## 导出文件命名
格式：`{客户公司名}-报关资料-{日期}.xlsx`
例如：`Nordic Tools AB-报关资料-2026-07-16.xlsx`

## 注意事项

1. **数据准确性**
   - 请在导出前仔细核对预览中的信息
   - 特别注意HS编码、品牌、型号等海关申报要素
   - 确认毛重、净重等物流信息准确

2. **必填字段**
   - 发货人税号：需要在系统中配置
   - HS编码：需要在产品信息中填写
   - 品牌型号：根据实际产品填写

3. **后续完善**
   - 导出后可在Excel中手动补充或修改信息
   - 建议保存一份模板以便重复使用
